创建用户引导——使用Intro.JS

Jul 19 2013

用户引导可以很好的向用户（新手）介绍网站、系统的功能分布和作用，同时当我们的应用（网站）每次改版增加新功能时，添加一个用户引导，可以让用户了解新功能。

用户引导有多种方式，常见的有两种：

  • 使用一系列图片和文字的方式，向用户介绍新功能，这个在移动APP中使用的相当多
  • 在每个功能区域用Tip提示的方式，就近介绍功能
    两种方式各有其优缺点和应用场景，这里主要就第二种方式进行说明。

下面是腾讯开放平台的一个新手引导的例子，是一种典型的提示方式的用户引导：

功能

采用提示方式的用户引导一般包含以下功能：

  • 可根据用户需求跳转到上一步、下一步、任意一步，并随时可中断
  • 可聚焦到当前步骤要说明的元素，并可通过添加边框或遮罩层突出该元素
  • 可自定义提示的内容和样式

  • 能够在步骤中进行简单的交互，可让用户进行实际操作
    开源世界里有许多专注于实现用户引导的轻量级JavaScript库供我们选用，根据自身的需要，我选取这些库应满足如下要求：

  • 易使用

  • 轻量级
  • 样式易定制
  • 可聚焦到元素，并呈现遮罩层
    于是最终选取了 Intro.js 这个库，这个库在 Github上也比较火，下面就用一个简单的例子用 intro.js 实现用户引导的功能。

实战

1. 引入文件

intro.js 不依赖其他任何库，因此只需要引用 intro.js 和 introjs.css即可。

2. 写代码

intro.js 提供两种方式配置流程：1. 直接写到html标签上，2. 使用JSON对象传入。在我们的实际开发中，优先使用第二种。

由于intro.js使用实在太简单了，我们平常使用就采用下面的代码就OK了。

    var intro = introJs();  // 实例化一个introJs对象
    intro.setOptions({  // 设置步骤
        nextLabel: “下一步<span></span>”,
        prevLabel: “上一步”,
        skipLabel: “跳过”,
        doneLabel: “完成”,
        steps: [{
            element: ‘#step1’,
            intro: “This is a tooltip.”
        }, {
            element: ‘#step2’,
            intro: “Ok, wasn’t that fun?”,
            position: ‘right’
        }, {
            element: ‘#step3’,
            intro: ‘More features, more fun.’,
            position: ‘left’
        }, {
            element: ‘#step4’,
            intro: “Another step.”,
            position: ‘bottom’
        }, {
            element: ‘#step5’,
            intro: ‘Get it, use it.’
        }]
    });

intro.start();  // 运行该引导</pre>


其他API的参考可参照项目主页：

https://github.com/usablica/intro.js

3. 后续的开发定制

可根据实际需要定制它的原生样式以符合具体需要，同时 intro.js 与 jQuery、Bootstrap等使用不会有任何冲突。并且在 Require.js、 Node.js 环境下也能良好运行。

值得注意的是，由于使用了 querySelector ，因此它不兼容IE7及以下的浏览器。
